BrowserBridge

Safari setup

Connect Safari to your private BrowserBridge server for tab handoff and shared BrowserBridge views.

Safari displays BrowserBridge data from your server. Native Safari bookmark/history upload is not available in this version.
History sync remains explicit and opt-in. BrowserBridge never syncs passwords, cookies, login sessions or form data.

Bookmark Sync

Safari support

Safari displays BrowserBridge bookmarks from your server and supports tab handoff.

Native Safari bookmark writing is not available in this Safari version. You can still view BrowserBridge bookmarks, search shared history, refresh server data, send tabs to Chrome, and receive tabs from Chrome.

Safe Folder Import: Not available for native Safari bookmarks

Merge: Not available for native Safari bookmarks

Mirror: Disabled for Safari
